About me

Filippo

Java & Web Developer

Programmo in Java dal 2021, principalmente lavorando su plugin Bukkit e bot Discord personalizzati con la libreria JDA. Negli anni ho affiancato a questo l'uso di librerie come Lombok e Sql2o, e sto muovendo i primi passi con Hibernate e Spring/Spring Boot: l'approccio non è troppo distante da come strutturo già i plugin e i database, quindi la curva di apprendimento è breve.

Mi posiziono come Mid-Level (sviluppatore intermedio), sebbene sto attualmente frequentando un corso ITS Umbria nel settore IT, tutte le mie conoscenze le ho acquisite da autodidatta, solo grazie a questo corso e ulteriori esperienze lavorative, mi hanno aiutato a esplorare il mondo del lavoro come developer, nonché ulteriori progetti personali che mostrerò di seguito.

Sul fronte web lavoro con HTML, CSS e JavaScript nativi, oltre a framework come Laravel e CodeIgniter, con Tailwind CSS per lo styling. Ho esperienza anche con Node.js e npm, usati attivamente nel progetto glitchvalley.it, un sito che combina Laravel, Node, npm e Tailwind.

Lavoro quotidianamente con Docker per isolare ambienti con librerie diverse da quelle installate, anche se per lo sviluppo preferisco un ambiente Ubuntu via WSL. Conosco bene gli ambienti di sviluppo professionali: uso IntelliJ IDEA Ultimate e PhpStorm con subscription JetBrains, e VS Code per modifiche rapide a file di configurazione.

EndlessHorizons Network

Online
IP glitchvalley.it
Versione 1.16.4 – 1.21.3
Modalità RolePlay

EndlessHorizons Network è un server Minecraft, costruito attorno a un'esperienza di RolePlay open-world da un team appassionato di sviluppo e di building da diverse parti d'Italia. In questa piattaforma i giocatori possono simulare la vita reale e viverla come preferiscono.

Dietro le quinte abbiamo sviluppato plugin Bukkit custom efficienti (linguaggio Java) per alimentare le meccaniche di gioco, gestito l'infrastruttura del server e costruito il sito ufficiale glitchvalley.it combinando Laravel, Node.js e Tailwind CSS. È il progetto che meglio racchiude tutte le mie competenze in un unico posto.

È proprio qua che nascondo il mio vero codice sorgente di tutto ciò che sviluppiamo (git della piattaforma). Come citato, tutto ciò che vedete nel mio Git personale è solo a scopo di divertimento e meme.
